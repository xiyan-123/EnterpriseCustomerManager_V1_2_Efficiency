企业客户管家 V1.3.1 多端备份互传与体验增强版

一、稳定基准
本版本基于“企业客户管家 V1.3 基础版 / V1.3 完整增强版 FIX1”继续增量升级。
保持包名不变：com.example.enterprisecustomer。
正常覆盖安装不会清空旧数据。请不要卸载旧版后再安装新版。

二、核心规则继续保持
1. 以企业为核心。
2. 企业名称是唯一识别标志。
3. 分组不参与唯一识别。
4. 企业名称相同则合并到同一家企业。
5. 联系人、电话、备注可以多次导入、不断积累。
6. 每次导入可以继续使用：联系人1、电话号码1、备注1。
7. 新电话号码追加，重复电话号码跳过。
8. 新备注追加，重复备注跳过。
9. 两次导入分组不同但企业名称相同，仍合并为同一家企业。
10. 已有分组不因 Excel 导入自动覆盖。
11. 导入通讯录名称为：公司名称-联系人姓名。
12. 只删除本 App 导入的手机通讯录联系人。
13. 后续升级不能破坏已导入数据。

三、客户状态规则
客户状态为：未设置 < 关注 < 跟进 < 潜力。
Excel 导入支持“客户状态”列。
高等级状态可以覆盖低等级状态，低等级默认不会覆盖高等级。
旧“重点”继续兼容为“潜力”。

四、V1.3.1 新增与优化功能
1. 底部导航调整为：首页｜客户｜跟进｜通讯录｜我的。
2. 首页升级为工作台：今日待跟进、逾期未跟进、即将跟进、未入库号码、客户概览和快捷操作前置。
3. 新增独立“跟进”页面：支持今日、逾期、即将、全部未完成跟进；支持完成、延期1天、延期3天、延期7天、查看企业和新增跟进。
4. 客户页增强：搜索支持备注和跟进内容；增加未导入通讯录、无电话、有电话、有备注、最近更新等筛选；批量操作收纳。
5. 企业详情页增加顶部总览卡：状态、分组、联系人数量、备注数量、已导入通讯录数量、最高号码星级、最近跟进和下次跟进。
6. 快速录入升级为智能粘贴录入：支持多电话识别、联系人电话配对、客户状态识别、标准日期识别、多个联系人预览编辑、跟进内容和下次跟进日期。
7. 完整备份增强：备份全部数据库，文件名包含企业数、电话数和时间。
8. 新增备份：导出上次基准时间之后新增或修改的数据，适合手机到平板/备用手机的增量互传。
9. 导入新增备份：以 Excel 方式导入并合并，不整体覆盖当前数据库。
10. 新增备份基准时间：可查看和手动设置新增备份起点。
11. 备份记录：记录完整备份、新增备份和完整恢复的时间、文件名、类型、结果和数据量。
12. 完整恢复安全增强：恢复前显示当前数据量，并自动备份当前数据；恢复后自动重建缓存和索引。
13. 导入日志增强：显示总行数、新增企业、合并企业、新增联系人、备注、标签、重复行、异常行等。
14. 删除企业前显示影响数量：联系人、备注、跟进记录、已导入通讯录联系人。
15. 号码质量清洗继续保留，清洗后自动重建缓存。
16. 我的页面模块化：分组管理、数据中心、备份互传、数据维护、日志与异常、系统说明、危险操作。
17. Excel 模板增强：增加客户来源、跟进方式、跟进内容、下次跟进日期等字段。

五、多设备互传建议
建议将一台设备作为主设备，例如手机。
首次同步平板：手机完整备份 → 平板完整恢复。
后续同步平板：手机新增备份 → 平板导入新增备份。
完整恢复会覆盖当前数据。
新增备份导入是合并数据，不会整体覆盖。
不建议手机和平板同时大量修改后互相导入。

六、推荐升级方式
1. 不卸载旧 App。
2. 直接安装 V1.3.1 APK 覆盖旧版。
3. 如果提示签名冲突或无法更新，不要卸载旧版，先保留旧版数据。
4. 进入 V1.3.1 后，建议先到“我的 → 备份互传 → 完整备份”导出一份完整备份。
