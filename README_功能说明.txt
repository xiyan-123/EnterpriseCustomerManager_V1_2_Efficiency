企业客户管家 V1.3.2 客户标签体系增强与多号码导入修正版

一、稳定基准
本版本基于 V1.3.1 多端备份互传与体验增强版继续升级，保持包名不变：com.example.enterprisecustomer。
正常覆盖安装不会清空旧数据。请不要卸载旧版后再安装新版。

二、核心规则保持不变
1. 以企业为核心。
2. 企业名称是唯一识别标志。
3. 分组不参与唯一识别。
4. 企业名称相同则合并到同一家企业。
5. 联系人、电话、备注可以多次导入、不断积累。
6. 每次导入可以继续使用：联系人1、电话号码1、备注1。
7. 新电话号码追加，重复电话号码跳过。
8. 新备注追加，重复备注跳过。
9. 两次导入分组不同但企业名称相同，仍合并为同一家企业。
10. 已有分组不因 Excel 导入自动覆盖。
11. 导入通讯录名称为：公司名称-联系人姓名。
12. 只删除本 App 导入的手机通讯录联系人。

三、V1.3.2 重点升级
1. 客户标签体系增强
- 新增正式标签表与企业-标签关联表。
- 自动迁移旧 companies.tags 标签字段。
- 保留旧 tags 字段兼容 Excel 导入导出和新增备份。
- 我的 → 标签管理：支持标签统计、查看客户、重命名、合并、删除。
- 客户页支持按标签筛选。
- 企业详情页标签按钮化，支持添加/移除标签。
- 批量操作支持批量添加标签、批量移除标签。
- 智能粘贴录入支持识别标签，并在预览页确认。
- 新增跟进记录时可同步添加快捷标签。

2. Excel 多号码导入修正
- 支持一个联系人对应多个手机号，例如：电话号码1 = 15026842995,13818630288。
- 导入后会拆成同一联系人名下的两条电话记录。
- 支持英文逗号、中文逗号、顿号、分号、斜杠、反斜杠、空格、换行等分隔符。
- 处理顺序为：先拆分、再清洗、再逐个去重、再逐个入库。
- 一个号码重复不会导致整个单元格跳过。
- 导入前预览会显示多号码单元格数量和拆分后电话数。

3. 阶段性稳定封版增强
- 增加 V1.3.2 启动显示页。
- 首次打开 V1.3.2 会自动生成升级前安全备份。
- 新增数据体检：检查企业、电话、标签、跟进等数据状态。
- 新增历史日志清理。
- 新增 90 天前未处理通话记录清理。
- 我的页面显示版本信息。

四、推荐升级方式
1. 不卸载旧 App。
2. 直接安装 V1.3.2 APK 覆盖旧版。
3. 首次打开后建议进入：我的 → 备份互传 → 完整备份，导出一份完整备份。
4. 重点测试：标签管理、按标签筛选、多号码导入、备份恢复。
