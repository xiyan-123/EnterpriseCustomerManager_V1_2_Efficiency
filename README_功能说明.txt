企业客户管家 V1.3 完整增强版

一、稳定基准
本版本基于 V1.2.1 可用版继续升级，保持包名不变：com.example.enterprisecustomer。
正常覆盖安装不会清空旧数据。请不要卸载旧版后再安装新版。

二、核心规则保持不变
1. 以企业为核心。
2. 企业名称是唯一识别标志。
3. 分组不参与唯一识别。
4. 企业名称相同则合并到同一家企业。
5. 联系人、电话、备注可以多次导入、不断积累。
6. 每次导入可以继续使用：联系人1、电话号码1、备注1。
7. 新电话号码追加，重复电话号码跳过。
8. 新备注追加，重复备注跳过。
9. 两次导入分组不同但企业名称相同，仍合并为同一家企业。
10. 已有分组不因 Excel 导入自动覆盖。
11. 导入通讯录名称为：公司名称-联系人姓名。
12. 只删除本 App 导入的手机通讯录联系人。

三、V1.3 新增功能
1. 数据备份与恢复
- 我的 → 数据维护 → 数据备份与恢复。
- 支持一键备份到 App 备份目录。
- 支持导出完整数据库备份文件。
- 支持从备份文件恢复。
- 恢复前会自动备份当前数据。
- 首次打开 V1.3 会自动生成升级前安全备份。

2. 导入前预览
- Excel 不再直接导入。
- 选择 Excel 后先解析预览。
- 预览内容包括：新企业、已存在企业、预计新增电话、重复电话、状态提升、异常行、空公司名称行、电话格式异常等。
- 用户确认后才正式写入数据库。

3. 客户状态增强
- 客户状态为：未设置、关注、跟进、潜力。
- 重要程度：未设置 < 关注 < 跟进 < 潜力。
- Excel 支持“客户状态”列。
- 导入页面支持本批次默认客户状态。
- 高等级状态可以覆盖低等级状态。
- 旧“重点”自动兼容为“潜力”。

4. 企业跟进记录
- 企业详情页新增“跟进记录”。
- 支持记录跟进方式：电话、微信、面访、短信、其他。
- 支持填写跟进内容和下次跟进日期。
- 首页显示今日待跟进、逾期未跟进。

5. 潜力客户快捷入口
- 首页新增“潜力客户”。
- 按潜力状态、3星号码、下次跟进、更新时间优先显示。

6. 通话记录生成跟进记录
- 通话记录匹配页面中，已匹配企业的通话记录可以生成跟进记录。
- 自动带入通话时间、通话类型和通话时长。

7. 疑似重复企业检测
- 我的 → 数据维护 → 疑似重复企业检测。
- 检测相同电话、企业名称高度相似等情况。
- 只提示，不自动合并。
- 合并必须手动确认。

8. 号码质量清洗
- 我的 → 数据维护 → 号码质量清洗。
- 检测号码格式问题、长度异常、同一号码出现在多个企业等。
- 清洗前必须确认。

9. 异常数据导出
- 我的 → 数据维护 → 导出异常数据 Excel。

四、推荐升级方式
1. 不卸载旧 App。
2. 直接安装 V1.3 APK 覆盖旧版。
3. 如果提示签名冲突或无法更新，不要卸载旧版，先保留旧版数据。
4. 进入 V1.3 后，先到“数据备份与恢复”导出一份完整备份。
